## 🚀 Overview

A persona-driven AI chatbot that allows users to interact with three distinct personalities inspired by real educators from Scaler/InterviewBit.

Built as part of a Prompt Engineering assignment, this project demonstrates how system prompts shape AI behavior and personality.

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```
git clone https://github.com/abhaysingh-oo7/solid-robot.git
cd solid-robot
```


## Backend Setup
```
cd server
npm install
npm start
```

## Create a .env file inside the server folder:
```
OPENROUTER_API_KEY=your_api_key_here
```

## Frontend Setup
```
cd client
npm install
npm run dev
```
